# Documentation, présentation et crédits en ligne du projet de jeu.

<!-- start-replace-subnav -->

Ce module explique comment documenter, présenter et publier un projet ludique de façon professionnelle : rédiger un `README` clair (objectif, instructions d'installation et d'exécution, format d'export web), fournir des captures d'écran et une courte vidéo démonstrative, ajouter un fichier `CREDITS.md` listant les auteurs et les licences des médias, et publier la version jouable sur une page publique (Itch.io, GitHub Pages). Il couvre également les éléments de présentation (pitch, identités visuelle et sonore), métadonnées essentielles (licence, contact, version), et vérifications finales avant diffusion (tests de compatibilité, affichage des crédits, conformité des licences, accessibilité basique). Le livrable attendu est une page publique accompagnée d'une documentation complète permettant à un tiers d'exécuter, comprendre et citer correctement le projet.
